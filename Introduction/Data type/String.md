### Char array for string

```cpp
char displayString[] = "Hello, World !";
cout << displayString;
```

### String handler

Get character index of a string

```cpp
string displayString = "Hello, World !";
cout << displayString[0]; //H
```

Add character to a string

```cpp
string displayString = "Hello, World ";
cout << displayString+'!'; //Hello, World !
```
